# Finnish Gift Tax Calculator

A Python program that calculates Finnish gift tax for gifts received from close relatives, based on the Finnish Tax Administration gift tax brackets (valid before 31 December 2025).

## Features

- Calculates tax for all gift tax bands
- Accepts decimal values
- Accepts comma-separated numbers (e.g. 75,000)
- Accepts space-separated numbers (e.g. 75 000)
- Formats tax amounts as currency

## Example

Input:

Value of Gift: €75,000

Output:

Amount of Tax: €7,100.00
